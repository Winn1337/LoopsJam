# LoopsJam
## Theme
Loop(s)
## Limitations
### Easy
The game must use only two buttons for controls
### Medium
The game must have multiple camera perspectives (first person, fixed, isometric etc.)
### Difficult
Weather or Environmental Effects that affect the game
